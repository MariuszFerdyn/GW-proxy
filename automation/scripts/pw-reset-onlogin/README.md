## Force password reset on login

### Usage

- Download the script
  ```bash
  wget https://raw.githubusercontent.com/k8-proxy/GW-proxy/master/automation/scripts/pw-reset-onlogin/pw-reset-onlogin.sh 
  ```

- Run the script
  ```bash
  ./pw-reset-onlogin.sh
  ```
- Optionally, delete the script
  ```bash
  rm pw-reset-onlogin.sh
  ```

- Logout `logout`
